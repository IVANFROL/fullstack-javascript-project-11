## RSS Агрегатор

RSS-агрегатор – сервис для агрегации RSS-потоков, с помощью которых удобно читать разнообразные источники.

### Установка

```bash
make install
```

### Запуск для разработки

```bash
make dev
```

### Сборка проекта

```bash
make build
```

### Линтер

```bash
make lint
```
